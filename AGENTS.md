# Project Instructions

## Project Shape

- This project is a static single-file HTML fan parody page.
- Main deliverable: `ubj-handsome-canvas-preview.html`.
- Keep changes dependency-free unless the user explicitly asks otherwise.

## Local Workflow

- Inspect the existing HTML/CSS/JS patterns before editing.
- Preserve the handmade MS Paint-style visual language, Korean copy tone, and embedded image assets.
- Prefer small, reviewable edits inside the existing file over introducing build tooling.

## Verification

- For content or UI edits, verify the HTML still parses and the edited interactions work in a browser when practical.
- Because there is no package manager or test suite, use focused static checks plus browser/manual verification.
